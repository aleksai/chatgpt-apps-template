# ChatGPT Apps SDK Template

## Requirements

Node.js 22.x

## Installation

```bash
git clone git@github.com:aleksai/chatgpt-apps-template.git
```

## Usage

```bash
cd chatgpt-apps-template/

npm i
npm run build

cd server/

npm start
```
